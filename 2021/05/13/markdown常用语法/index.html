分类于 hexo 阅读次数： Valine： 本文字数： 1.6k

刚搭建好个人博客环境，第一篇博客就写一下markdown的常用语法吧，以前不写博客所以用得少，就当复习了。
markdown是一种纯文本格式的标记语言，可以完全脱离鼠标完成格式排版，学会之后能够大大提高写作效率，而且学习成本比较低，几分钟就够了。

不同平台对markdown的支持有些微区别，注意区分
